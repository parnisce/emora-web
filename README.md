# Emora Mood Tracker Website

A premium, modern multi-page website for the **Emora Mood Tracker** app. This website showcases the app's features, provides company information, and handles user inquiries and subscriptions.

## 🚀 Vision
Emora is designed to help users understand their emotional journey with simple, intuitive mood tracking and deep insights. This website serves as the primary gateway for new and existing users to engage with the Emora ecosystem.

## 📄 Pages
- **Home**: Introducing Emora with premium app mockups and value propositions.
- **About Us**: Our mission and the story behind the brand.
- **FAQ**: Answers to common questions about privacy, features, and usage.
- **Blog**: Educational content on mental health and mood patterns.
- **Manage Subscription**: Transparent pricing tiers (Free vs. Premium).
- **Contact Us**: Direct line for support and inquiries.

## 🛠️ Tech Stack
- **Core**: Vanilla HTML5 / CSS3 / JavaScript
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Design**: Custom CSS system with "Emora Green" branding
- **Typography**: Outfit (Headings) & Inter (Body) from Google Fonts

## 💻 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher recommended)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
To start the local development server with hot-reload:
```bash
npm run dev
```

### Production Build
To generate a production-ready bundle in the `dist` directory:
```bash
npm run build
```
